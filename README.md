EVA战斗座舱HUD

目前完美支持车型：W205 C63s AMG - 7速版，全部DID ECU数据可用，动态请求模式最高可达30HZ刷新率。
同时支持AMG GT系列车型（C190），唯一BUG是读不到变速箱档位，这个是AMG GT自己的内部映射问题毕竟就他一个7DCT，可以用计算档位模式弥补。

原生开发硬件是基于OBDLINK MX+ 和 CX这两款插头进行开发的，MX+是传统SPP蓝牙OBD插头，性能最强，但是不支持iOS，CX是新款BLE5.1插头，支持IOS但是延迟会比较大。
APP原生做了对OBDLINK插头的特殊指令优化，如果你没有这个牌子的插头也没关系，做了降级通道，就算普通ELM327应该也能跑到10hz这样，通过30FPS插帧引擎应该不影响使用效果。

默认带的参数上下警告数据仅供参考，自己根据真实车况改一下。

In the end，这只是因为热爱的个人作品，希望你们也能喜欢，并且由衷的希望能在这个时代为所有使用者，带去哪怕一点纯粹的快乐。

本项目采用flutter编译，测试机型为小米17PRO手机，其他机型的兼容性问题可以反馈到这里，记得带截图。

GT7特别版 - 更新日期 3月6日
----------
本版本仅支持PS5游戏跑车浪漫旅7，即GT7，采用UDP遥测方案，特别感谢Github上的snimat和bornhall两位作者整理出来的UDP遥测算法。
如无意外，本版本将会是对GT7支持的最终版本，除非有什么好的idea整合进去，否则不会对GT7特别版维持更新。
有关对iRacing，AC，ACC，LMU的支持，还需要很多时间，因为作者已经把严肃的模拟赛车戒了。

功能特性
GT7 实时遥测：通过局域网接收并解密 PlayStation 广播的 UDP 数据流（Salsa20 解密，刷新率约 60Hz）。
EVA 风格 HUD：深度还原 NERV 视觉风格（含六角网格与 CRT 扫描线特效）。主界面实时显示车速、档位、转速、油门/刹车踏板开度及当前油量。
动态赛道小地图：小地图以半透明形式融入档位背景层，并支持动态缩放（直道自动缩小视野，弯道自动放大细节）。
圈速对比 (Delta Time)：通过位置匹配算法，实时计算并显示当前圈与最佳圈的时间差。
轮胎与动态监控：提供四轮独立温度的分段可视化展示，实时检测车辆滑移状态，并同步指示 TCS/ABS 介入及轮胎过热（TYRE HOT）警告。
智能燃油管理：采用 EMA 算法平滑计算单圈油耗，实时预估剩余可跑圈数 (Fuel Laps) 并判断燃油余量状态。
自定义警报系统：内置 6 通道独立警报，支持自定义触发阈值。包含“非常事態”、“冷却過熱”、“増圧限界”、“燃圧喪失”等多重 NERV 风格全屏视觉警告叠层。
同步换挡提示：精准同步 GT7 游戏内的转速警告 (Rev Warning) 与红线断油 (Rev Limiter)，提供清晰的升档指示（支持独立开关）。
驾驶风格统计：记录并分析操作数据，直观展示全油门、全刹车、空挡滑行及左脚刹车的操作占比。
个性化配置：支持用户在本地目录自定义替换背景壁纸、开关BGM 及系统提示音效。
